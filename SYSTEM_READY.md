## 🎯 How to Use the System

### 1. Register Employees
1. Go to http://localhost:3000
2. Click "Register Employee"
3. Fill in employee details (ID, name, email, department)
4. Upload a photo (optional)
5. Submit

### 2. Mark Attendance
1. Click "Mark Attendance"
2. Select "Manual Entry"
3. Enter Employee ID
4. Click "Check In" or "Check Out"

### 3. View Reports
1. Go to "Admin Dashboard"
2. View system statistics
3. Check attendance records
4. Generate reports

## 🔧 Optional: Face Recognition Setup

To enable automatic face recognition for attendance:

1. **Install CMake:**
   - Download from https://cmake.org/download/
   - Add to system PATH during installation

2. **Install Python Libraries:**
   ```bash
   python -m pip install dlib face_recognition opencv-python
   ```

3. **Restart Server:**
   ```bash
   npm start
   ```
